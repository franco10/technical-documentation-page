Introduction

Python is an interpreted high-level general-purpose programming language. Its design philosophy emphasizes code readability with its use of significant indentation. Its language constructs as well as its object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects.

Python is dynamically-typed and garbage-collected. It supports multiple programming paradigms, including structured (particularly, procedural), object-oriented and functional programming. It is often described as a "batteries included" language due to its comprehensive standard library.

The language's core philosophy is summarized in the document The Zen of Python (PEP 20), which includes aphorisms such as:

  • Beautiful is better than ugly.
  • Explicit is better than implicit.
  • Simple is better than complex.
  • Complex is better than complicated.
  • Readability counts.

Python was designed by Guido van Rossum.

Variables

In Python there are different types of variables.

Integer number:

Int, or integer, is a whole number, positive or negative, without decimals, of unlimited length.

int_variable = 12

Floating-point number:

Float, or "floating point number" is a number, positive or negative, containing one or more decimals.

int_variable = 12.5

String:

Strings are arrays of bytes representing Unicode characters.

str_variable = "Hello world"

Boolean value:

It is a variable that can only take two values: true or false.

bool_value_1 = True
bool_value_1 = False

Set:

A set is an unordered collection of items. Every set element is unique (no duplicates) and must be immutable (cannot be changed).

aset = {'blue', 'red', 'yellow'}

List:

List items are ordered, changeable, and allow duplicate values.

alist = ['blue', 'red', 'yellow', 'blue']

Tuple:

Tuple items are ordered, unchangeable, and allow duplicate values.

atuple = ('blue', 'red', 'yellow', 'blue')

Dictionary

Dictionaries are used to store data values in key:value pairs.
Dictionary items are ordered, changeable, and does not allow duplicates.

adict = {

'firstname': 'Franco',

'lastname': 'Salinas',

'course': 'Web development',

'division': 2

}
Print and Input

print function

Used to print on screen.

There are different ways to use this function.

print("Value 1:", value_1, "\nValue 2:", value_2) print("Value 1: {}\nValue 2 {}".format(value_1, value_2)) print(f'Value 1: {value_1}\nValue 2: {value_2}') print("Value 1: " + str(value_1) + "\nValue 2: " + str(value_2))

input function

It is used for the user to enter values.

variable = input("Enter a value: ")
Control structures

There are multiple types of control structures.

if

Allows the program to branch based on the evaluation of an expression.

if expression:

statement 1

elif:

statement 2

else:

statement 3

while

With the while loop we can execute a set of statements as long as a condition is true.

while condition == True:

statement 1

else:

statement 2

for

A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string).

for i in alist:

print(i)
